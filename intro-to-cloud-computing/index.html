Understanding Cloud Computing Concepts

IaaS, PaaS, SaaS & Top Cloud Providers

Presented by: Laxman Poudel

Introduction to Cloud Computing

  • Delivery of computing services over the internet.
  • Key Services: Servers, Storage, Databases, Networking, Software, Analytics, Intelligence.
  • Benefits: Cost-effective, Scalable, Reliable, Fast, and Secure.

Service Models Overview

  • IaaS: Infrastructure as a Service
  • PaaS: Platform as a Service
  • SaaS: Software as a Service

IaaS (Infrastructure as a Service)

  • Virtualized computing resources via the internet
  • Users manage: OS, Storage, Apps
  • Providers manage: Servers, Networking, Virtualization
  • Examples: Amazon EC2, Google Compute Engine, Azure VMs

PaaS (Platform as a Service)

  • Platform for developers to build applications
  • Users manage: Applications, Data
  • Providers manage: OS, Middleware, Runtime
  • Examples: Google App Engine, Azure App Services, AWS Elastic Beanstalk

SaaS (Software as a Service)

  • Ready-to-use applications over the internet
  • Managed completely by the provider
  • Users need only a web browser
  • Examples: Gmail, Microsoft 365, Dropbox, Salesforce

Comparison Table

Feature IaaS PaaS SaaS
Control Level High Medium Low

Top Cloud Providers

  • AWS: Market leader, extensive services, global reach
  • Azure: Strong enterprise integration, hybrid capabilities
  • GCP: Strong in AI/ML, Big Data

Conclusion

  • Cloud computing is revolutionizing IT
  • Select the right model for your needs
  • Top providers offer scalable, innovative services
